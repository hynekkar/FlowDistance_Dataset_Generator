# FlowDistance Dataset Generator

```
usage: main.py [-h] -i INPUT -o OUTPUT -c CONTENT

Generate dataset for flow distance research

options:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Path to file with input url
  -o OUTPUT, --output OUTPUT
                        Output directory for packet captures
  -c CONTENT, --content CONTENT
                        Output directory for downloaded content
```
